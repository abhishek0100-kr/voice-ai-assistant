# AI Attribution

## Overview

Project Aether is an original software engineering project designed and implemented by the author. Artificial Intelligence services and AI-assisted development tools were used to accelerate development, improve productivity, and enhance documentation quality.

The project's architecture, implementation, testing, debugging, and final integration were completed by the author.

---

# AI Services Used

## Google Gemini Live API

The primary conversational engine of Project Aether is powered by Google's Gemini Live API.

It is responsible for:

- Processing real-time streaming audio
- Understanding spoken user input
- Generating conversational responses
- Streaming synthesized voice output

---

## Google Gemini 2.5 Flash

Gemini 2.5 Flash is used as the fallback conversational engine.

When the persistent Live API connection becomes unavailable, the application automatically switches to the REST API to maintain uninterrupted user interaction.
